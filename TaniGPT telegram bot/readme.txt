# 🤖 TaniGPT Telegram Bot

TaniGPT is a friendly, Hinglish-speaking AI chatbot powered by **Mistral AI** and built for **Telegram** users. Created with ❤️ by [Tanishk Sharma](https://tanishksharma.dev), the bot offers engaging, real-time conversations with a unique vibe.

---

## 🚀 Features

- 👋 Interactive onboarding with name and phone number
- 🧠 Chat powered by Mistral Large Language Model
- 📞 Phone number verification and user indexing
- 🛠️ Admin panel (Telegram + Web) to manage users and history
- 💾 Local JSON storage for fast, easy data handling
- 🎨 Hinglish tone and emoji-powered responses

---

## 🛠️ Tech Stack

- **Python 3.10+**
- **[python-telegram-bot](https://github.com/python-telegram-bot/python-telegram-bot)**
- **[Flask](https://flask.palletsprojects.com/)**
- **[Mistral AI API](https://mistral.ai/)**
- **dotenv** for secrets
- **JSON** files as a lightweight database
